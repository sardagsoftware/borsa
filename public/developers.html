SDK Kütüphaneleri

Favori programlama diliniz için resmi SDK'larımızı kullanın

TypeScript / Node.js

OpenAPI Generated

TypeScript desteği ile tam tip güvenli Node.js SDK. Express, Next.js ve diğer framework'lerle uyumlu. OpenAPI spesifikasyonundan otomatik üretilir.

npm install @lydian/api-client
OpenAPI Spec İndir

Python

OpenAPI Generated

Async/await desteği ile modern Python SDK. Django, Flask ve FastAPI ile tam entegrasyon. OpenAPI spesifikasyonundan otomatik üretilir.

pip install lydian-api-client
OpenAPI Spec İndir

Go

OpenAPI Generated

Yüksek performanslı Go SDK. Concurrency desteği ve minimal memory footprint ile üretim ortamları için optimize edilmiş. OpenAPI spesifikasyonundan otomatik üretilir.

go get github.com/lydian/api-client-go

🔐 Authentication Methods

LyDian Platform üç farklı authentication yöntemi sunar. İhtiyaçlarınıza göre en uygun yöntemi seçin.

API Key

Recommended for most use cases

En basit ve hızlı authentication yöntemi. Dashboard'dan API key alın ve hemen kullanmaya başlayın. SHA256 hash ile güvenli, rate limiting ve usage tracking dahil.

// HTTP Header
X-API-Key: lyd_1234567890abcdef
✅ Simple setup
✅ Quick start
✅ Rate limited
✅ Usage tracking

OAuth2 / JWT

For user-facing applications

Kullanıcı bazlı authentication için ideal. Token-based sistem, Google/GitHub OAuth2 entegrasyonu, refresh token desteği ve granular scope yönetimi.

// HTTP Header
Authorization: Bearer eyJhbGc...
✅ User scopes
✅ Refresh tokens
✅ OAuth2 flows
✅ Granular access

HMAC-SHA256

Enterprise-grade security

En yüksek güvenlik seviyesi. Request imzalama, replay attack koruması, timestamp validation ve body integrity kontrolü. Kurumsal uygulamalar için önerilir.

// HTTP Headers
X-Signature: sha256=abc123...
X-Timestamp: 1234567890
✅ Request signing
✅ Replay protection
✅ Body integrity
✅ Maximum security
