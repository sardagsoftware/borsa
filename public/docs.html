Giriş

LyDian, Microsoft Azure, Google Gemini ve Advanced AI Platform Enterprise AI modellerini birleştiren kurumsal düzeyde bir AI platformudur. Tek bir API üzerinden 23 farklı AI modeline erişim sağlar.

Platform Özellikleri
  • 23 AI modeli (GPT-4, Enterprise AI, Gemini, DALL-E)
  • 84 dil desteği Z.AI dil paketi ile
  • Gerçek zamanlı streaming ve WebSocket
  • Çok modlu işleme (metin, görsel, ses)
  • Kurumsal güvenlik ve ölçeklenebilirlik

Hızlı Başlangıç

Platformu kullanmaya başlamak için sadece birkaç basit adım:

JavaScript 
// API anahtarınızla isteği yapın
const response = await fetch('http://localhost:3100/api/chat', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    message: 'Merhaba yapay zeka!',
    model: 'gpt-4'
  })
});

const data = await response.json();
console.log(data.response);
Python 
import requests

response = requests.post('http://localhost:3100/api/chat',
  headers={
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  json={
    'message': 'Merhaba yapay zeka!',
    'model': 'gpt-4'
  })

print(response.json()['response'])

Streaming API

Gerçek zamanlı yanıtlar için streaming desteği.

JavaScript 
const response = await fetch('http://localhost:3100/api/chat', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    message: 'Uzun bir makale yaz',
    stream: true
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const chunk = decoder.decode(value);
  console.log(chunk); // Her kelime geldiğinde göster
}
