Geliştirici merkezi

LyDian AI'yı uygulamalarınıza entegre edin. Kapsamlı SDK'lar, API dokümantasyonu ve kod örnekleri ile dakikalar içinde başlayın.

import { LyDian } from '@lydian/sdk'; const client = new LyDian({ apiKey: 'sk-lydian-...' }); const response = await client.chat.create({ model: 'lydian-turbo', messages: [{ role: 'user', content: 'Merhaba!' }] }); console.log(response.content);

SDK Kütüphaneleri

Favori programlama diliniz için resmi SDK'larımızı kullanın

Node.js

v2.1.0

TypeScript desteği ile tam tip güvenli Node.js SDK. Express, Next.js ve diğer framework'lerle uyumlu.

npm install @lydian/sdk
İndir

Python

v2.0.5

Async/await desteği ile modern Python SDK. Django, Flask ve FastAPI ile tam entegrasyon.

pip install lydian
İndir

Go

v1.8.2

Yüksek performanslı Go SDK. Concurrency desteği ve minimal memory footprint ile üretim ortamları için optimize edilmiş.

go get github.com/lydian/sdk-go
İndir

Hızlı Başlangıç

5 dakikada ilk AI entegrasyonunuzu yapın

1

API Anahtarı Alın

Dashboard'dan API anahtarınızı oluşturun. Her proje için ayrı anahtarlar kullanabilirsiniz.

# Dashboard'da API Keys bölümünden yeni anahtar oluşturun API_KEY="sk-lydian-..."
2

SDK'yı Kurun

Favori programlama diliniz için SDK'yı paket yöneticiniz ile kurun.

# Node.js npm install @lydian/sdk # Python pip install lydian # Go go get github.com/lydian/sdk-go
3

İlk Çağrınızı Yapın

Birkaç satır kod ile AI modellerine erişin ve yanıt almaya başlayın.

const { LyDian } = require('@lydian/sdk'); const client = new LyDian({ apiKey: process.env.LYDIAN_API_KEY }); async function main() { const response = await client.chat.create({ model: 'lydian-turbo', messages: [{ role: 'user', content: 'LyDian API nasıl kullanılır?' }] }); console.log(response.content); } main();
4
